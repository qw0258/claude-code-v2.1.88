## 正文

Anthropic 在 npm 发布 Claude Code 时忘了去掉 source map，被人还原出了 1884 个源文件。我仔细读了一遍，发现了 7 个从没公开过的隐藏功能：

1️⃣ BUDDY — AI 电子宠物（见 P2）
终端里养拓麻歌子！18 种物种（鸭子、龙、幽灵、卡比巴拉...），稀有度从普通 60% 到传说 1%，还有 1% 闪光概率。用你的账号哈希生成，每人独一无二，改配置都没用。预计 5 月上线

2️⃣ KAIROS — 永不关机的 Claude（见 P3）
关掉终端 Claude 还在！跨会话持久运行，每天写日志，晚上自动"做梦"整理记忆。还有主动模式——没人说话它自己找活干，没活就 Sleep 等着。命令超 15 秒自动丢后台

3️⃣ ULTRAPLAN — 云端 30 分钟深度规划（见 P4）
本地一键把难题甩给云端 Opus 4.6，远程独立研究最长半小时。在浏览器里看方案、改方案、批准执行，还能"传送"回本地跑

4️⃣ Coordinator — 多 Agent 编排模式（见 P5）
主 Claude 变成纯指挥官，手下只有"派活、通信、停工"三个按钮。Worker 并行干活汇报结果，核心铁律：禁止甩锅式委派

5️⃣ 26+ 隐藏命令 & 秘密开关（见 P6）
/buddy pet 摸宠物、/dream 手动做梦、/teleport 传送会话、/fork 分叉 Agent... 还有一堆秘密环境变量和 CLI 参数

6️⃣ Bridge — 手机/网页远程遥控终端（见 P7）
从 claude.ai 或手机直接操控本地 CLI，WebSocket 实时连接。还有跨 Claude 会话的进程间通信

7️⃣ 32 个编译开关 + 远程门控（见 P8）
我们用的外部版本是阉割版！Anthropic 内部用 USER_TYPE=ant 解锁全部功能，还有一堆 GrowthBook A/B 测试开关在远程控制

数据来源：@anthropic-ai/claude-code v2.1.88 npm 包的 cli.js.map
还原后的完整源码仓库：github.com/pengchengneo/Claude-Code
